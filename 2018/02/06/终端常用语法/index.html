终端常用语法

Veröffentlicht am 2018-02-06

终端常用语法

  1. ps -e 查看每个模块的全路径
  2. ssh root@ip 使用ssh 登录手机
  3. scp 需要拷贝的文件 要拷贝的文件路径
  4. cycript -p TargetApp 对TargetApp进行注入
  5. image list -o -f 查看APP加载模块
  6. debugserver *:1234 -a "App" 开启APP的1234端口的监听
    a. debugserver -x backboard *:1234 路径
  7. grep -r 检索条件 路径 对某个路径下进行条件检索

lldb 使用

  1. process connect connect://iOSip:3333 针对手机上ip的3333端口进行lldb
  2. br s -a address 针对address下断点
  3. br dis 禁用断点
  4. br en 开启断点
  5. br del 删除断点
  6. br com add 1 针对1号断点进行命令追加,使用DONE结束追加命令,当断点1倍出发时候会调用追加的命令
  7. p 打印出对象的类型,值,变量名 例如 (long) $2 = 1324,可以强行更改对象的类型
  8. po 打印对象的值
  9. p/x 以16进制的方式打印地址
  10. x/10 打印出10个bite的数据
  11. ni 不进入函数下一步
  12. si进入函数下一步
  13. register write 寄存器 = 1 expression 寄存器 = 1 两个命令一样,修改寄存器的值为1
  14. register read 寄存器 expression 寄存器 读取寄存器
  15. frame info 查看当前栈帧函数名,如果有参数包括参数
  1. frame variable ,查看当前栈帧函数名,参数,函数内部局部变量
  1. frame select 0 展示选择栈帧时部分代码
  1. thread backtrace -c 5 在当前线程中选择前5 个栈帧展示
  1. disassemble --frame 反汇编当前栈帧di -f
  1. disassemble --name test 反汇编test方法 di -n test
  1. image lookup -v --address 0x0000000106dfe000 查看0x0000000106dfe000这个模块上的详细信息

#cycript 中语法

  1. [[[UIApplication sharedApplication]keyWindow] recursiveDescription] [[UIApp keyWindow] recursiveDescription]相同效果,展示当前界面下所有视图结构

  2. [self.view recursiveDescription] view的视图结构
